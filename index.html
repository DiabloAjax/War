Как пользоваться

Назначение

Приложение предназначено для распределения игроков по 12 комнатам, быстрой корректировки составов, поиска, импорта списков и выгрузки результата в текст или изображение. Все изменения сохраняются автоматически и восстанавливаются при следующем открытии файла.

Структура экрана

На десктопе слева расположен пул свободных игроков, справа комнаты. У пула есть поиск, кнопка загрузки, кнопка справки и кнопка скрытия. У блока комнат есть отдельный поиск, кнопка копирования всех комнат и кнопка красивого снимка. На мобильных пул открывается отдельным нижним модальным окном по кнопке Пул игроков.

Работа с игроками

Игроков можно перетаскивать из пула в комнаты, между комнатами и обратно в пул. Если перетащить игрока на другого игрока внутри комнаты, они поменяются местами. Кнопка ↩ на карточке в комнате сразу возвращает игрока в пул.

Быстрые действия мышью

Двойной клик по карточке игрока в пуле автоматически отправляет его в ближайшую по порядку комнату. Двойной клик по игроку в комнате возвращает его обратно в пул. Во время перетаскивания работает автопрокрутка страницы, а внутри пула отдельно работает внутренний автоскролл.

Редактирование игрока

Нажмите ✎ на карточке, чтобы открыть редактирование. В режиме редактирования можно изменить имя игрока и выбрать один из 7 стандартных цветов. Сохранение выполняется кнопкой ✓ или клавишей Enter. Пустое имя сохранить нельзя. Пока карточка редактируется, быстрые действия по двойному клику и drag для неё отключаются.

Быстрый перенос по списку

Рядом с кнопкой редактирования в карточке есть выпадающий список комнат. Если выбрать в нём номер комнаты, игрок сразу переносится в выбранную комнату. Это работает и для игроков в пуле, и для игроков, уже находящихся в комнатах.

Поиск

Поиск по комнатам и поиск по пулу работают независимо. Совпадения подсвечиваются прямо в имени игрока. Поддерживается поиск по похожему вводу латиницей и кириллицей. При вводе интерфейс автоматически прокручивается к первому совпадению. Крестик × очищает поиск, а Esc очищает активное поле.

Комнаты и заполненность

У каждой комнаты есть визуальное состояние: пустая, частично заполненная, почти полная и полная. В счётчике отображается количество игроков в формате текущее/8. Если игроков меньше 8, текущее число становится красным. Переполнение тоже разрешено: игроки могут переноситься в уже заполненные комнаты, и счётчик может стать, например, 9/8 или больше.

Якоря комнат

В мобильной версии слева отображаются фиксированные якоря комнат: 1-1, 1-2 и так далее. Нажатие на якорь прокручивает страницу к нужной комнате. Цвет якоря показывает состояние комнаты: пустая — серый, полная — зелёный, переполненная — красный.

Импорт из Excel и CSV

Кнопка Загрузить открывает выбор файла. Обрабатывается первый лист и первая колонка. Если имя уже существует, игрок сохраняется без изменений. Если имени нет, создаётся новый игрок в пуле. Если в текущем приложении есть игроки, которых нет в загруженном файле, они удаляются полностью, включая удаление из комнат. Новые импортированные игроки получают оранжевый цвет по умолчанию.

Копирование состава

Кнопка ⧉ в верхней панели копирует все комнаты сразу. Кнопка ⧉ внутри комнаты копирует только состав этой комнаты. При копировании имена автоматически преобразуются в формат @ник без пробелов. После копирования кнопка и соответствующий блок кратко подсвечиваются. На мобильных используется усиленный fallback для буфера обмена.

Красивый снимок

Кнопка ✦ создаёт аккуратный PNG-снимок блока комнат. В экспортном режиме из снимка убираются служебные элементы интерфейса: поиск, кнопки в карточках, выпадающие списки и лишние управляющие элементы. В заголовках комнат на снимке остаются только номера комнат без слова Комната.

Пул на мобильных

На телефоне пул открывается нижней кнопкой Пул игроков. Внутри окна работает отдельный скролл. Чтобы начать перенос игрока, нужно не просто провести пальцем, а коротко удержать карточку. После успешного удержания приложение даёт лёгкий вибро-отклик, окно пула уходит с экрана, а под пальцем появляется карточка-призрак. Комната под пальцем подсвечивается заметнее, чтобы было проще отпустить игрока в нужное место.

Скрытие пула и сохранение

На десктопе кнопка ‹ сворачивает пул игроков, а › возвращает его обратно. При скрытии пула область комнат расширяется. Все действия сохраняются автоматически: переносы, перестановки, редактирование имени, смена цвета, импорт из файла и переполненные комнаты. При повторном открытии страницы приложение восстанавливает последнее сохранённое состояние.

Горячие клавиши

Enter сохраняет имя игрока в режиме редактирования. Esc закрывает окно справки или очищает активное поле поиска. Остальные действия выполняются мышью или касанием: drag-and-drop, двойной клик, кнопки на карточках и выпадающий список комнат.

Выберите файл настроек

Чтобы автосохранение работало, выберите существующий файл или создайте новый.
